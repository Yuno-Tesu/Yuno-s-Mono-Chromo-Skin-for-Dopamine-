# Installing Theme

To install the theme:
- Open Dopamine 3
- Open the <span style="color:blue;">Menu</span>
- Open <span style="color:blue;">Settings</span>
- Under the Section **Colors**, Press <span style="color:blue;">Add More Themes</span>
- Copy the theme file into the directory (folder) that opened up

May have to restart Dopamine 3.
